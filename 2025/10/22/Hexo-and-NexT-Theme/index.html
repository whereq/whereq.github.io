9.1 常见问题

  1. 主题不生效：

    • 检查 theme: next 配置
    • 运行 hexo clean

  2. 页面 404：

    • 检查菜单配置的路径
    • 确认页面已创建

  3. 样式异常：

    • 检查自定义 CSS 语法
    • 确认字体文件路径

10. 最佳实践

  1. 版本控制：将整个 Hexo 项目推送到一个单独的仓库
  2. 备份配置：定期备份主题配置文件
  3. 渐进式配置：一次只修改一个功能，测试后再继续
  4. 使用 CDN：对于静态资源使用 CDN 加速
